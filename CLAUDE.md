# 短篇小说创作项目

这是一个基于 Claude 的短篇小说创作辅助项目。

## 可用技能

### short-novel-workflow

短篇小说创作工作流技能，提供完整的10步创作引导流程。

**触发方式：**
- 说 "创作小说" 或 "开始创作"
- 说 "短篇小说创作"
- 说 "帮我写一个故事"

**工作流程：**
1. 选择受众（男频/女频）
2. 填写创作想法
3. 选择题材
4. 设置目标字数
5. 生成故事背景设定
6. 生成角色设定
7. 生成故事情节大纲
8. 选择小说名称
9. 完成小说内容
10. 保存小说

## 项目结构

```
短篇小说创作/
├── .claude/
│   └── skills/
│       └── short-novel-workflow.md  # 创作工作流技能
├── novels/                           # 创作完成的小说存放目录
├── templates/                        # 可选的模板文件
├── README.md                         # 项目说明
└── CLAUDE.md                         # 项目配置（本文件）
```

## 创作规范

- 目标字数：默认 20,000 字（可调整）
- 支持题材：历史、玄幻、都市、奇幻、仙侠、科幻、衍射、言情、悬疑、现实、体育、其他
- 受众类型：男频、女频
