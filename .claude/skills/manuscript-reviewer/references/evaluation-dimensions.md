## 1. 影响力 Impact (22%)

**定义**：作品引发读者情感反应和留下深刻印象的能力。

**核心问题**：
- 这个故事是否让我有所感受？
- 是否值得我投入时间阅读？
- 是否会在读完后留下深刻印象？

**评估要点**：
- 首段吸引力：是否引人入胜、不同凡响
- 情感共鸣：是否能激发读者的情感反应
- 记忆点：是否有让人印象深刻的场景或细节
- 张力发展：是否建立并维持叙事张力

**常见问题**：
- 首段吸引力不足（39%退稿原因）
- 缺乏张力
- 刻意制造"惊喜"结局
- 滥用戏剧性元素

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 首段立即抓住读者，情感共鸣强烈，读后难忘 |
| 7-8 | 有吸引力，能引发一定情感反应 |
| 5-6 | 平淡无奇，难以产生共鸣 |
| 1-4 | 毫无吸引力，无法继续阅读 |

## 2. 创意性 Creativity (15%)

**定义**：作品在主题、叙事、语言等方面的原创性和想象力。

**核心问题**：
- 是否富有想象力、原创性？
- 是否对熟悉事物提供新视角？
- 是否拓展了文学可能性的理解？

**评估要点**：
- 主题新颖度：是否探索新题材或提供独特切入角度
- 叙事创新：是否尝试创新的叙事手法
- 语言创新：是否有独特的语体风格

**常见问题**：
- 主题俗套（三角恋、穿越、重生等）
- 语言缺乏新意
- 过度依赖陈词滥调、副词堆砌
- 情节可预测

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 主题/叙事/语言均有创新，令人耳目一新 |
| 7-8 | 有一定新意，但不突破常规 |
| 5-6 | 中规中矩，无明显创新 |
| 1-4 | 俗套、陈旧、可预测 |

## 3. 语言/文笔 Language (15%)

**定义**：作品的语言运用能力和文字表达水平。

**核心问题**：
- 语言是否娴熟运用？
- 表达是否清晰？
- 是否展现抒情性？

**评估要点**：
- 清晰度：表达是否准确、易懂
- 抒情性：语言的美感和韵律
- 创新性：独特的词汇使用和句式

**常见问题**：
- 语法错误
- 句式冗长
- 词汇有限
- 形容词过度堆砌（AI特征）

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 语言精炼优美，有独特风格 |
| 7-8 | 表达清晰，有一定文采 |
| 5-6 | 基本通顺，偶有瑕疵 |
| 1-4 | 错误多，表达不清 |

## 4. 格式规范 Guidelines (14%)

**定义**：作品是否符合投稿指南和排版规范。

**核心问题**：
- 是否遵循投稿指南？
- 排版是否整洁专业？

**评估要点**：
- 文档格式：Word/文本格式规范
- 段落结构：清晰分段，信息量适中
- 基础信息：作者联系方式、字数统计完整
- 标题设计：是否具有吸引力

**常见问题**：
- 长篇大论不分段
- 文件命名不规范
- 缺少作者信息
- 标题不吸引人

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 完全符合规范，排版专业美观 |
| 7-8 | 基本符合规范，小问题不影响阅读 |
| 5-6 | 有明显格式问题，影响阅读体验 |
| 1-4 | 格式混乱，不符合基本要求 |

## 5. 润色 Polish (7%)

**定义**：作品的完成度和打磨程度。

**核心问题**：
- 是否经过精心照料、打磨？
- 完成度如何？

**评估要点**：
- 校对质量：错别字、病句数量
- 细节处理：标点、格式细节
- 整体完成度：是否接近可发表状态

**常见问题**：
- 校对疏漏
- 表达粗糙
- 明显的未完成感

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 精心打磨，接近完美 |
| 7-8 | 基本完成，小瑕疵可忽略 |
| 5-6 | 有明显疏漏，需进一步润色 |
| 1-4 | 粗糙未完成，需大量修改 |

## 6. 结构 Structure (8%)

**定义**：作品的叙事逻辑和节奏控制。

**核心问题**：
- 叙事是否引导读者？
- 视角、开头和结尾是否有效？

**评估要点**：
- 叙事逻辑：时间线、因果关系是否清晰
- 节奏控制：情节密度是否张弛有度
- 开篇处理：是否快速进入核心冲突
- 结尾处理：是否有力、令人满意

**常见问题**（54%退稿原因）：
- 叙事混乱（时序不清、视角不合理）
- 节奏失衡
- 开头/结尾处理不当
- 开篇铺垫过长

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 结构精巧，节奏完美 |
| 7-8 | 结构清晰，节奏基本合理 |
| 5-6 | 结构有问题但不影响理解 |
| 1-4 | 结构混乱，难以跟随 |

## 7. 深度 Depth (6%)

**定义**：作品的人物内心世界和主题挖掘程度。

**核心问题**：
- 人物是否有完整的内心世界？
- 主题是否触及人性深层问题？

**评估要点**：
- 人物动机：是否合理且复杂
- 内心描写：是否有具体的心理活动
- 人物形象：是否避免刻板化
- 主题深度：是否触及人性或社会问题

**常见问题**（56%退稿原因，最高）：
- 表层信息，缺乏深入
- 角色动机薄弱
- 缺乏内心描写
- 人物形象刻板化

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 人物立体深刻，主题有哲学高度 |
| 7-8 | 有一定深度，人物有内心世界 |
| 5-6 | 浅层叙事，人物单薄 |
| 1-4 | 完全缺乏深度，人物工具化 |

## 8. 简洁性 Economy (5%)

**定义**：作品是否充分利用每一个字，无冗余。

**核心问题**：
- 是否可以用更少字数表达同样意思？
- 是否存在可删除而不损害效果的内容？

**评估要点**：
- 无冗余段落
- 无重复词句
- 字字有用

**常见问题**（14%退稿原因）：
- 冗长、重复
- 8000字的故事可能需要压缩到3800字

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 精炼紧凑，一字不可删 |
| 7-8 | 基本简洁，偶有冗余 |
| 5-6 | 有明显冗余内容 |
| 1-4 | 严重冗长，需大幅压缩 |

---

## 9. 真实性 Authenticity (4%)

**定义**：作品的情感真实性和自然度。

**核心问题**：
- 是否感觉自然真实？
- 是否展现情感诚实？

**评估要点**：
- 情感真实性：人物情感是否自然流露
- 逻辑合理性：细节是否符合常识
- 整体质感：是否有"人味儿"

**重要原则**：事实真实次于情感真实

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 情感真实动人，完全有"人味儿" |
| 7-8 | 基本真实，偶有做作 |
| 5-6 | 情感不够自然 |
| 1-4 | 虚假做作，明显AI特征 |

---

## 10. 主题 Subject (3%)

**定义**：作品题材的独特性和普世意义。

**核心问题**：
- 题材是否独特但主题是否具有普世性？
- 是否触及大的人类问题？

**评估要点**：
- 题材独特性：是否选择独特的职业/背景
- 普世意义：是否触及普遍人性

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 题材独特且主题深刻 |
| 7-8 | 有一定独特性 |
| 5-6 | 常见题材，处理得当 |
| 1-4 | 题材俗套，无新意 |

---

## 11. 语调 Voice (2%)

**定义**：作品的独特声音和一致性。

**核心问题**：
- 声音是否独特？
- 是否与叙述者/人物一致？

**评估要点**：
- 声音独特性：是否有辨识度
- 一致性：是否与人物设定匹配

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 声音独特，高度一致 |
| 7-8 | 基本一致，有个性 |
| 5-6 | 语调平淡 |
| 1-4 | 语调混乱，不一致 |

---

## 12. 对话 Dialogue (1%)

**定义**：作品对话的质量和与叙述的配合度。

**核心问题**：
- 对话是否与叙述配合或促进叙述？
- 格式是否正确？

**评估要点**：
- 对话自然度：是否生活化
- 对话功能：是否推动情节或刻画人物
- 格式正确：标点、分段是否规范

**评分标准**：
| 分数 | 描述 |
|-----|------|
| 9-10 | 对话生动自然，推动叙事 |
| 7-8 | 对话基本自然 |
| 5-6 | 对话有些生硬 |
| 1-4 | 对话虚假，格式错误 |
