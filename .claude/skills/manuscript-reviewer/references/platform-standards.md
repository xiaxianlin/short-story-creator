## 传统文学期刊

### 核心标准
- **文学性优先**：遵循"故事优先"原则
- **内容重要性高于文笔**：故事价值决定稿件命运
- **思想深度**：重视立意和社会意义

### 审稿周期
- 电子稿：1个月
- 纸质稿：2个月
- 整体范围：1-3个月

### 字数偏好
- 标准范围：4000-8000字
- 最佳长度：4500字

### 审稿特点
1. **三审三校制度**严格执行
2. 重视作品的政治导向、思想倾向
3. 关注艺术创新性
4. 编辑细致审读，周期较长

### 权重分配
| 维度 | 权重 |
|-----|------|
| 深度 | 56% |
| 结构 | 54% |
| 影响力 | 22% |
| 创意性 | 15% |

### 代表平台
- 《人民文学》
- 《收获》
- 《十月》
- 《短篇小说》
- 《小说月报》

## 网络文学平台

### 核心标准
- **市场性导向**：注重商业价值
- **节奏为王**：开篇即决战
- **爽点密集**：满足读者期待

### 审稿周期
- 番茄小说：2-7天
- 起点文学：3-7天
- 晋江文学城：3-7天

### 字数偏好
- 短篇：1-5万字
- 可连载形式

### 2025年最新变化
| 平台 | 变化 |
|-----|------|
| 番茄小说 | 短篇稿费翻倍至100元/千字，爆款可参与500万奖金池 |
| 起点文学 | 3月起拒绝AI稿，只接受100%人工创作 |
| 晋江文学 | 分级管控AI辅助写作 |

### 审稿特点
1. **首400字决策**：前三章完读率决定流量分配
2. AI检测成为必备环节
3. 标题/节奏/爽点是核心指标
4. 日更能力是重要考量

### 权重分配
| 维度 | 权重 |
|-----|------|
| 影响力 | 22% |
| 创意性 | 15% |
| 格式规范 | 14% |
| 结构 | 8% |

### 代表平台
- 番茄小说
- 起点中文网
- 晋江文学城
- 七猫小说

## 新媒体故事号

### 核心标准
- **流量导向**：打开率决定收益
- **标题为王**：标题直接决定审稿结果
- **三观正确**：避免引起争议

### 审稿周期
- 公众号：3-7天
- 征稿函响应：较快

### 字数偏好
- 标准范围：8000-15000字
- 最佳长度：10000-12000字

### 审稿特点
1. **标题首要**：不起眼的标题直接被拒
2. **节奏快速**：碎片化阅读场景
3. **"苏感"人设**：让读者心动的人物设定
4. **反转设计**：情节需要有惊喜
5. **调性匹配**：需符合平台既往风格

### 核心指标
| 指标 | 说明 |
|-----|------|
| 标题吸引力 | 决定打开率 |
| 情节新颖性 | 有看点 |
| "苏感"人设 | 高富帅、可爱等 |
| 反转设计 | 情节惊喜 |
| 三观正确 | 符合社会主流 |

### 退稿常见原因
1. 标题不吸引人
2. 节奏过慢
3. 不符合平台调性
4. 三观有问题

### 代表平台
- 各大公众号故事号
- 小红书故事类账号
- 抖音故事类账号

## 知乎盐选

### 核心标准
- **完整故事性**：故事必须完整
- **改编潜力**：适合影视/音频改编
- **专业性**：内容需有专业背景支撑

### 审稿周期
- 7-15天

### 字数偏好
- 标准范围：1-3万字
- 最佳长度：2万字以上

### 2025年数据
- 2万字以上故事，累计分账破2000万案例出现

### 审稿特点
1. 重视故事的完整性
2. 关注改编潜力
3. 专业性内容加分
4. 配图审稿日益重要

### 权重分配
| 维度 | 权重 |
|-----|------|
| 完整性 | 高 |
| 改编潜力 | 高 |
| 专业性 | 中 |
| 影响力 | 中 |

## 国际文学杂志

### 核心标准
- **多元化**：不同文化背景有不同侧重
- **国际化**：跨文化视野

### 通用核心维度
1. 影响力 (Impact)
2. 创意性 (Creativity)
3. 语言/文笔 (Language/Prose)

### 差异化特点
| 地区 | 特点 |
|-----|------|
| 欧洲杂志 | 更注重实验性和形式创新 |
| 美国商业杂志 | 更看重故事娱乐性和可读性 |
| 英美文学杂志 | 平衡文学性与可读性 |

### 特别注意事项
1. **格式规范严格**："遵循投稿指南"提及率14%
2. **非母语作者**：语言特色可能被视为优势
3. **文化差异**：需注意文化背景表达

### 代表平台
- The New Yorker
- Granta
- The Paris Review
- Ploughshares

## 平台选择决策指南

```
你的作品特征：
├── 追求文学性、思想深度
│   └── 选择：传统文学期刊
│
├── 追求市场性、商业变现
│   ├── 节奏快、爽点多
│   │   └── 选择：网络文学平台
│   │
│   └── 情感向、反转设计
│       └── 选择：新媒体故事号
│
├── 专业性强、有改编潜力
│   └── 选择：知乎盐选
│
└── 国际化视野、实验性强
    └── 选择：国际文学杂志
```

## 投稿前检查清单

### 通用检查
```
□ 研究目标平台的样刊/样文
□ 了解平台的审美取向
□ 确认字数符合要求
□ 格式规范检查
□ 基础校对（错别字/病句）
```

### 平台特定检查

**传统期刊**：
```
□ 故事是否有深度
□ 立意是否独特
□ 文学性是否足够
```

**网络平台**：
```
□ 首章是否有吸引力
□ 节奏是否够快
□ 是否有爽点设计
□ AI检测是否通过
```

**新媒体**：
```
□ 标题是否吸引人
□ 是否有反转设计
□ 三观是否正确
□ 是否符合平台调性
```

**知乎盐选**：
```
□ 故事是否完整
□ 是否有改编潜力
□ 专业内容是否准确
```
