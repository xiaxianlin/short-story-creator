# 长篇小说故事素材库

## 简介

本素材库是从长篇小说中提炼的可复用故事素材合集，旨在为创作提供参考和灵感。素材聚焦于**情节设计的亮点**和**可借鉴的创新点**，而非照搬原著内容。

## 🎯 使用指南

### 1. 查找素材
- 通过`素材索引.md`快速浏览所有素材
- 按分类目录查找特定类型的素材
- 根据关键词搜索相关素材

### 2. 理解素材
- 阅读"核心剧情"了解故事主线
- 查看"可借鉴元素"获取可复用的设计点
- 参考"适用题材"判断是否符合你的创作方向

### 3. 应用素材
- **情节混搭**：组合不同素材的元素创造新故事
- **人设改造**：在经典人设基础上增加新特征
- **背景移植**：将成功的世界观应用到新题材
- **冲突升级**：在经典冲突上增加新矛盾维度

## 💡 分类说明

### 核心剧情模板
包含完整的、可独立复用的故事主线结构，如：
- 穿越者权臣崛起
- 气运流复仇逆袭
- 凡人流谨慎修仙

### 情感冲突类型
聚焦人物关系和情感张力的设计，如：
- 父子情深与生存博弈
- 父子双穿的互动关系

### 经典桥段
高辨识度的经典场景和情节片段，如：
- 学院流厚颜无耻开局
- 梦境现实双重世界

### 人物关系模型
特色角色设定和互动模式，如：
- 超然智者与帝王博弈
- 特色人设的塑造方法

### 世界观构建
背景设定和体系设计的参考，如：
- 儒道流文以载道体系
- 蛮荒流人族崛起格局

## ⚠️ 使用原则

1. **参考而非抄袭**：素材仅供创作参考，请勿直接照搬原著情节
2. **组合创新**：建议将多个素材元素组合，创造新的故事
3. **尊重版权**：使用素材时请注意尊重原著版权
4. **标注来源**：如有借鉴，建议标注灵感来源
